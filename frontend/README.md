# LeadsForge Frontend

React + Vite + TypeScript.

## Setup

```bash
npm install
```

## API base URL

By default the app calls `http://127.0.0.1:8000`. To point at another backend:

```bash
cp .env.example .env.local
# edit VITE_API_BASE_URL
```

## Run

```bash
npm run dev
```

## Checks

```bash
npm run lint
npm run build
```
